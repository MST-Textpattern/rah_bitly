h1. Extending rah_bitly

p. This directory contains modules for rah_bitly.

h2. Prevent

p. This is an example plugin for rah_bitly which showcases extending. The plugin will prevent generating Bitly links for articles published in section which name starts with underscore (@_@).

h2. Localhost

p. Appends @http://example.com/#@ to the published short link's location. This makes a permanent link pointing to a local address publishable.
